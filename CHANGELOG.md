# TechWords 版本变更记录

本文档记录每次正式版本的功能、数据结构、词库、缓存、测试与回滚信息。

版本发布时必须同步更新：

1. `index.html` 中的 `APP_VERSION`
2. `manifest.json` 中的 `version`
3. `sw.js` 中的 `CACHE_NAME`
4. 本文件中的版本记录
5. `README.md` 中的当前版本与使用说明

---

## v1.1.1 — 2026-06-22

### 版本定位

增强训练理解、训练中查词、今日学习回顾和品牌图标展示，不改变现有单文件架构、词库格式及每日学习状态机。

### 训练内容与发音

- 例句填空题在英文挖空句下方显示该例句的完整中文翻译。
- 英文单词题干可点击朗读。
- 英文例句题干可点击朗读完整原句。
- 中选英、听音选词和例句填空的英文选项可点击朗读。
- 拼写题中的字母或短语词块点击时可朗读。
- 保留答对或答错后朗读正确目标词的反馈。
- 对正确英文选项避免重复朗读；错误英文选项先朗读所选内容，再朗读正确目标词。

### 训练中查看单词

- 每个训练题增加“查看单词”按钮。
- 可直接进入当前单词的完整详情页，查看释义、大白话、技术场景、科普、搭配、易混点和例句。
- 从训练进入详情时隐藏编辑和删除操作，避免训练会话中的词条被意外改动。
- 返回详情前的原训练题，保留题目、作答状态和当前进度，不重新生成训练页面。

### 今日已学

- 首页增加“查看今日已学”入口。
- 仅展示当天学习流程中已经实际接触过的新词；尚未打开的预选新词不计入。
- 进入训练阶段或完成学习后，展示当天全部新词。
- 今日已学列表可进入单词详情，并返回原列表。

### App 图标

- 使用用户提供的新图案替换 192×192 和 512×512 的 PWA/iPhone 图标。
- 移除 manifest 对旧 `icon.svg` 的优先引用，统一使用新版 PNG 图标。
- 首页顶部中央增加去除白底和外部空白后的透明主体 Logo。
- Service Worker 缓存名称升级为 `techwords-v1.1.1`，确保已安装设备获取新图标和 Logo。

### 版本与兼容性

- App版本：`1.1.1`
- 数据结构版本：`4`（未变更）
- Service Worker缓存：`techwords-v1.1.1`
- 未改变 `words-data.js` 的全局数组格式、词条ID或本地存储键。

### 回归测试

- `index.html` 内联脚本语法检查通过。
- `manifest.json` 解析检查通过。
- Service Worker列出的本地缓存资源完整性检查通过。
- 首页透明 Logo 加载成功，PNG尺寸为512×512，四角透明。
- “今日已学 → 单词详情 → 返回今日已学”链路通过。
- “训练题 → 查看单词 → 返回原训练题”链路通过，题目和进度未重置。
- 训练详情中的编辑和删除操作已隐藏。
- 例句填空完整中文翻译显示测试通过。
- 英文题干和英文选项点击发音事件绑定测试通过。
- 英文选项作答、正确答案反馈和下一题按钮测试通过。
- 浏览器运行期间未发现应用控制台错误。

## v1.1.0 — 2026-06-22

### 版本定位

重构完整学习流程，将“浏览词卡、主动训练、历史复习”拆分为清晰的每日学习状态机。

### 首页与每日任务

- 首页拆分为“开始学习”和“开始复习”两个独立入口。
- 每天首次开始学习时固定选择当日新词，默认10条。
- 当日多次进入、刷新或退出后继续时，词条及顺序保持不变。
- 学习中途显示“继续学习”或“继续训练”。
- 当日学习完成后，“今日学习已完成”按钮禁用，不能重复开始正式学习。
- 到期历史词汇独立进入复习队列。
- 复习完成后，“今日复习已完成”按钮禁用。
- 第二天自动建立新的学习会话，并根据 `nextReview` 生成历史复习任务。

### 新词学习流程

- 第一阶段为“认识新词”：
  - 展示单词、音标、发音、释义、大白话、例句和标签。
  - 只显示“上一个 / 下一个”。
  - 最后一张显示“开始训练”。
  - 首次接触单词时不再要求选择“认识 / 模糊 / 不认识”。
- 第二阶段自动编排题型，用户不能手动切换训练模式。
- 每个新词至少进入两种训练题型。

### 自动训练

- 自动穿插：
  - 英文选中文
  - 中文选英文
  - 拼写或短语排序
  - 听音选词
  - 例句填空
- 答对或答错后都会自动朗读一次正确英文单词。
- 答错立即展示正确答案和解释。
- 错题自动追加到本轮后部再次出现。
- 学习完成页显示正确率和错题次数。

### 历史复习

- 只选择以前学习过且已经到期的词汇。
- 当天刚学习的新词不会立即混入复习入口。
- 新词完成首次训练后，第一次复习安排在次日。
- 后续复习继续使用间隔重复调度。
- 历史错题优先排序。

### 每日会话数据

- 新增 `tw_daily_session`：
  - 日期
  - 当日固定新词ID
  - 认识阶段进度
  - 自动训练任务和进度
  - 当日固定复习词ID
  - 错题重试任务
  - 学习和复习完成状态
- JSON导出现在包含当日会话。
- 导入同日备份时可恢复当日进度。
- 数据结构版本升级为 `schemaVersion: 4`。

### 词库搜索

- 修复搜索结果被标签大量干扰的问题。
- 精确英文词条匹配时只显示精确结果。
- 排序优先级：
  1. 英文精确匹配
  2. 英文前缀
  3. 英文包含
  4. 中文释义
  5. 大白话、场景解释和搭配
  6. 标签
- 搜索时自动切换到“全部”筛选。
- 增加搜索结果数量。
- 增加清空搜索按钮。
- 对大小写、空格、连字符、下划线和点号进行兼容处理。

### PWA与缓存

- App版本：`1.1.0`
- 数据结构版本：`4`
- Service Worker缓存：`techwords-v1.1.0`

### 回归测试

- 每日新词固定队列测试通过。
- 中途退出后继续学习测试通过。
- 首次学习“上一个 / 下一个”流程测试通过。
- 自动20题训练流程测试通过。
- 学习完成后按钮锁定测试通过。
- 独立历史复习及完成锁定测试通过。
- 错题重新入队测试通过。
- 搜索 `LLM` 精确返回1条结果。
- 搜索清空和7个筛选按钮测试通过。
- 新增、收藏、发音、编辑、删除、删除取消测试通过。
- 设置步进器和显示开关测试通过。
- 数据导出触发成功提示。
- JSON导入、预置词合并、设置恢复和数据版本迁移测试通过。
- 底部导航和统计页测试通过。
- 页面控制台无错误。
- 390×844 iPhone视口回归通过。

### 回滚

- 回滚时必须同时恢复：
  - `index.html`
  - `manifest.json`
  - `sw.js`
  - `CHANGELOG.md`
- 如从v1.1.0回滚，旧版本会忽略 `tw_daily_session`，但建议先导出JSON备份。

## v1.0.1 — 2026-06-22

### 版本定位

从基础词汇卡片升级为面向技术人员的离线英语学习与 AI 概念科普 App。

### 词库变化

- 原始预置词库：500条。
- 删除37条确认重复记录。
- 新增34条 AI 行业术语。
- 当前预置词库：497条。
- 保留 `pipeline`、`hook`、`environment` 三组不同技术语境的同形词条。
- 新增词条包括：
  - LLM、GLM、MoE、SLM、VLM、MLLM
  - active parameters、dense model、model size、VRAM
  - FP16、BF16、INT8、INT4
  - reasoning model、base model、instruct model、open-weight
  - distillation、synthetic data、test-time compute
  - agent loop、context engineering、context compaction、context rot
  - computer use、vibe coding、AI slop、AI wrapper、copilot
  - jailbreak、grounding、reranker、vector store

### 词条内容升级

- 增加可选字段：
  - `plainMeaning`：大白话解释
  - `contextMeaning`：技术场景解释
  - `knowledge`：相关科普
  - `collocations`：常见搭配
  - `confusions`：易混点
- 重点增强：
  - `parameter`
  - `context window`
  - `quantization`
  - `harness`
- 增加7B、70B、参数量、模型大小、显存、量化精度、MoE总参数与激活参数等知识。

### 学习与训练

- 保留卡片复习。
- 新增英文选中文。
- 新增中文选英文。
- 新增单词拼写和短语排序。
- 新增听音选词。
- 新增例句填空。
- 答题后显示正确答案和通俗解释。
- 错题会记录并提高后续复习优先级。
- 统计页增加训练正确率和错题词数。

### 复习算法修复

- 修复新词选择“模糊”后退出学习队列的问题。
- 修复“不认识”词未到期可能立即再次出现的问题。
- “不认识”：6小时后复习。
- “模糊”：24小时后复习。
- “认识”：按3、7、14、30、90天推进。
- 新增 `attemptCount` 和 `lastReviewedAt`。

### 数据迁移

- 数据结构版本升级为 `schemaVersion: 3`。
- 新增 `tw_meta`。
- 旧版500词数据自动合并重复记录。
- 自动补入新增 AI 词条。
- 自动补入旧词条缺失的科普字段。
- 保留用户学习状态、收藏、复习次数和用户自定义词条。
- 导入旧备份后也会执行迁移。

### 日期与分类

- 日期统计改用设备本地日期，避免中国时区凌晨错日。
- 统一来源与标签选项。
- 搜索支持标准释义、大白话、场景解释、搭配和标签。

### PWA与缓存

- App版本：`1.0.1`
- 数据结构版本：`3`
- Service Worker缓存：`techwords-v1.0.1-build2`
- 缓存策略改为网络优先、离线回退。
- 在线启动时主动检查Service Worker更新。

### 测试结果

- 497个唯一词条ID。
- 所有必填词条字段完整。
- 旧版500词迁移测试通过。
- 用户自定义词条保留测试通过。
- 学习状态与收藏迁移测试通过。
- 六种训练模式交互测试通过。
- GLM、参数量和显存科普展示测试通过。
- 训练正确率与错题统计测试通过。
- 390×844 iPhone视口回归通过。
- 页面控制台无错误。
- `index.html`、`sw.js`、`manifest.json`语法检查通过。

### 回滚

- 原始500词备份：`words-data.backup.js`
- 回滚整个版本时需要同时恢复：
  - `index.html`
  - `words-data.js`
  - `sw.js`
  - `manifest.json`
- 仅恢复 `words-data.js` 不足以完整回滚，因为数据迁移和训练逻辑位于 `index.html`。

### 已知限制

- 数据仍存储在浏览器 `localStorage`。
- 词库页面仍一次渲染全部词条。
- 发音依赖iOS/Web Speech系统语音。
- 当前部署脚本的中文提示可能乱码，但部署命令本身不受影响。
